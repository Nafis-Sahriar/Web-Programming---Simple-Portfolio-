# Kaniz Roksana Riya Portfolio

This portfolio has been converted into a framework-free static website.

## Run

Open `index.html` directly in any modern browser.

No installation, build step, React, Next.js, JSX, TypeScript, npm package, or bundler is required.

## Files

- `index.html` contains the semantic portfolio sections.
- `style.css` contains custom animations and reusable styles.
- `script.js` contains vanilla JavaScript for icons, smooth scrolling, back-to-top behavior, and the footer year.
- `assets/` contains the preserved images copied from the original project.
